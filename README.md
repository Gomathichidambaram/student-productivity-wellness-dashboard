Student Productivity & Mental Wellness Analytics Dashboard

📊 Project Overview
This Power BI dashboard analyzes student productivity, stress levels, sleep habits, and screen time patterns to identify behavioral trends and lifestyle impacts.

🎯 Objectives
- Track daily study hours and stress patterns
- Analyze relationship between screen time and productivity
- Visualize mood distribution
- Provide actionable lifestyle insights

🛠 Tools Used
- Power BI
- DAX Measures
- Data Visualization Techniques

 📈 Key Insights
- Higher screen time correlates with reduced productivity
- Increased sleep improves focus and lowers stress
- Balanced routines lead to better mood distribution.
